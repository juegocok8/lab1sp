🏢 Sistema de Cálculo de Póliza de Seguro de Auto

Desarrollo con React · Almacenamiento en memoria · Sin base de datos

Java Swing → React

Una compañía aseguradora desea desarrollar un sistema que permita calcular automáticamente el costo anual de una póliza de seguro de automóvil, considerando diferentes factores de riesgo del conductor y del vehículo. El sistema se desarrollará con React utilizando componentes funcionales, hooks y almacenamiento en memoria mediante arreglos.

Objetivo del Sistema

  1. Calcular el valor anual de la póliza
  2. Mostrar el detalle de los recargos y descuentos aplicados
  3. Guardar cada cotización en memoria
  4. Generar estadísticas generales de las pólizas registradas

Cálculo de Prima Base

Prima Base = 5% del valor comercial del vehículo

Ejemplo:

Vehículo valorado en $12,000 → 12,000 × 0.05 = $600

Factores que Modifican la Póliza

A

Edad del Conductor

  • Menor de 25 años → +20%
  • Entre 25 y 60 años → 0%
  • Mayor de 60 años → +15%
B

Tipo de Vehículo

  • Sedán → 0%
  • SUV → +10%
  • Pickup → +8%
  • Deportivo → +25%
C

Historial de Accidentes

  • 0 accidentes → -10%
  • 1 accidente → +15%
  • 2 accidentes → +30%
  • 3 o más accidentes → +50%
D

Uso del Vehículo

  • Personal → 0%
  • Comercial → +20%
E

Zona de Circulación

  • Zona normal → 0%
  • Zona de alto riesgo → +15%
F

Promoción Aleatoria

Generar número aleatorio del 1 al 5

Si el número es 3 → aplicar descuento especial del -12%

Resultado Final

  • Mostrar prima base
  • Mostrar total de recargos
  • Mostrar total de descuentos
  • Mostrar prima anual final
  • Calcular cuota mensual (Prima anual / 12)
  • Clasificar riesgo: Bajo, Medio, Alto o Crítico

Clasificación de Riesgo

Prima Anual FinalClasificación
Menor a $500Bajo
$500 - $900Medio
$901 - $1500Alto
Mayor a $1500Crítico

Ejemplo práctico:

Valor vehículo: $10,000 → Prima base: $500 | Recargos: $200 | Descuentos: $50

Prima final: $650 → Rango $500–$900 → Riesgo = Medio

Reporte General

  • 1. Total de pólizas registradascontador
  • 2. Promedio de primasmedia
  • 3. Tipo de vehículo más aseguradomoda
  • 4. Cliente con la póliza más altamáx
  • 5. Conductores con descuento por buen historial (0 accidentes)contador
  • 6. Clientes con recargo por alto riesgo (riesgo Alto o Crítico)contador

Validaciones Obligatorias

  • No permitir edad menor de 18 años
  • No permitir valor del vehículo menor o igual a cero
  • No permitir año del vehículo menor a 1990
  • No permitir nombre vacío
⚛️ Desarrollo con React
Implementación con componentes funcionales, useState para manejo de estado, useEffect para efectos secundarios.
Almacenamiento en memoria mediante arreglos. No se utiliza base de datos. Tabla equivalente a JTable usando componentes de Bootstrap.
